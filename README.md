# 🤖 Career Compass AI

Career Compass AI is a simple web-based chatbot developed as part of the **Lenovo LEAP AI Internship**. The chatbot is designed to help students with career guidance, resume preparation, interview tips, internship guidance, and skill development through a clean and user-friendly interface.

---

## 📌 Project Overview

Career Compass AI provides students with quick access to career-related information using a simple chatbot interface. It demonstrates the basic concepts of chatbot development using front-end web technologies.

---

## ✨ Features

- 💼 Career Guidance
- 📄 Resume Building Tips
- 🎤 Interview Preparation
- 🚀 Skill Development
- 🏢 Internship Guidance
- 🎓 Higher Studies Information
- 📱 Responsive User Interface
